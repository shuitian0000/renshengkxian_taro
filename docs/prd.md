# 人生K线图微信小程序需求文档

## 1. 应用概述

### 1.1 应用名称
人生K线图

### 1.2 应用描述
一款基于八字命理的运势分析微信小程序，用户输入姓名八字信息后，系统调用大模型生成人生K线图和命理报告，支持报告保存、查看历史和PDF导出功能。\n
## 2. 核心功能

### 2.1 八字信息输入
- 用户输入姓名、地区，地区精确到地级市
- 用户输入出生年月日时信息
- 支持公历（阳历）和农历（阴历）切换
- 自动进行公历与农历的正确转换

### 2.2 用户可选输入正面头像
- 用户可选输入面相
- 面相可以上传照片或拍摄照片，同时提示用户授权相册或相机，并完善小程序代码审核时需要的用户隐私保护指引
- 面相照片需要检测正面头像，免冠照片，面部，头部，耳朵等清晰可见
- 调用大模型提取面部相关特征信息，用于后续的命理理论

### 2.3 人生K线图生成
- 调用大模型根据姓名、八字信息，调用大模型，综合运用传统阴阳五行，干支历法及其符号系统，传统八字命理运势，易经占卜等多类信息，生成人生K线图数据。同时也要综合所在地区特点，未来社会和国家或者地区的发展趋势，综合进行预测，生成人生K线图数据。\n- K线图坐标：Y轴为年龄（虚岁，5岁间隔），X轴为评分（0-10分）\n- 每年K线包含：开盘、收盘、最高、最低、得分、吉凶趋势描述
- 大运周期用紫色虚线分隔，标注大运干支
- K线图与输入八字信息相匹配

### 2.4 命理报告生成
- 包含命理总评、性格分析、事业分析、风水建议、财富分析、婚姻分析、健康分析、六亲分析等维度
- 每个维度包含分析文案和评分（0-10分）\n- 报告结构清晰，各维度独立呈现

### 2.5 报告保存与管理
- 微信账号快捷登录
- 登录后可保存报告记录
- 登录后支持查看历史报告
- 登录后支持下载导出PDF格式报告
- 未登录用户无法保存报告，未登录无法查看历史报告，未登录无法下载报告，触发相关操作时提示登录

### 2.6 报告导出
- 导出PDF文件名格式：姓名拼音_生成日期_命理报告.pdf
- PDF内容包含完整命理报告和K线图

### 2.7 K线图交互
- 支持全屏查看
- 自动适配手机横屏/竖屏
- 支持退出全屏
- K线图批注默认隐藏，点击K线显示

### 2.8 重新推演
- 在K线图页面提供重新推演选项
- 使用已输入信息重新计算，无需用户重新输入

### 2.9 本地算法框架
- 精心设计小程序本地模型和算法框架，包括提取面相特征的算法，包括未来地区和国家社会的发展趋势
- 在无网络或AI服务不可用时使用本地算法生成K线图
- 基于用户出生年份生成数据，保证逻辑一致性

## 3. 视觉风格

### 3.1 设计理念
极简东方哲思+静谧神秘意境，传递天人合一的极简哲思，适配现代UI交互逻辑

### 3.2 色彩体系
- 主色调：玄黑（#1A1A1A）、宣纸白（#F5F0E6）\n- 辅助色：鎏金（#D4AF37）\n- 功能色：吉-鎏金、凶-暗红（#8B0000）、中性-浅灰（#B3B3B3）\n- 禁用其他色彩、渐变、纹理叠加

### 3.3 字体体系
- 全程使用思源宋体
- 标题：加粗思源宋体
- 正文：常规思源宋体（14px，行高22px）\n- 摒弃装饰性字体变形

### 3.4 动效设计
- 按钮点击：轻微缩放反馈（缩放比例0.98，时长200ms）\n- 页面切换：淡入淡出（时长300ms）\n- 下拉刷新：极简太极旋转（1s/圈）\n- 摒弃氛围动效、透明度变化、光影变化

### 3.5 排版与元素
- 极简对称布局，主次分明
- 致极留白，页面边缘留白≥32px（移动端）/64px（PC端）\n- 模块间距统一48px
- 核心信息突出，辅助信息极简呈现
- 摒弃八卦方位分布、复杂装饰元素、材质纹理
- 交互动效简洁，无多余动态干扰
